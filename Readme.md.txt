Este programa está diseñado en JavaScript para ayudarte a hacer cotizaciones de servicios de manera sencilla. La idea es que puedas ingresar diferentes propiedades y tus ingresos, y el programa te dará una cotización basada en esos datos. Lo mejor es que puedes hacer varias cotizaciones hasta que decidas salir.

¿Cómo funciona?

1. Cuando inicias el programa, te da la bienvenida y te explica brevemente qué hacer.
2. Te pide que ingreses una propiedad y tus ingresos.
3. Luego, el programa calcula la cotización usando las reglas que hemos definido.
4. Después, te muestra el resultado de la cotización.
5. Si quieres, puedes hacer otra cotización o salir, simplemente escribiendo 'Salir'.

Mejoras que se podrían hacer

- Crear una interfaz gráfica para que sea más fácil de usar.
- Hacer validaciones más estrictas en las entradas del usuario, para evitar errores.
- Permitir exportar las cotizaciones a archivos CSV o PDF.
- Usar una base de datos para guardar las cotizaciones que se generen.
- Incluir análisis de datos para ofrecer recomendaciones personalizadas.
